You started a json server when running npm run dev. You can check the API is available at localhost:4000.
Your first mission is to get the cupcakes from the API.
Add an instance of HttpClient in the api-service and fetch the localhost:4000/cupcakes endpoint.
In the CupcakeList component ( src/app/pages/cupcake-list), use the service function value on ngOnInit and subscribe the function. Then, get response in an property.
You can check the result with a console.log for this step.

Hint You will need a property to store the cupcakes. Use an empty array as initial value.

üç™ Step 3 - Get the accessories

For this step, get the accessory list from the API.
As below, Add an instance of HttpClient in the api-service and fetch the localhost:4000/cupcakes endpoint.
In the CupcakeList component ( src/app/pages/cupcake-list), use the service function value on ngOnInit and subscribe the function. Then, get response in an property.
Once again, you can check the result with a console.log.

Hint You will need an other property to store the accessories (initialized with an empty array).
